# Code Review Guide

This project is a monorepo containing a Rust/WASM core, Elm frontend, Node.js backend, and Chrome Extension.

## ðŸŸ¢ Source Code (Please Review)

### Core Logic (Rust & WASM)
- **`src/`**: The main Rust crate (`proto-rust`). Contains domain models, API definitions, and WASM exports.
    - `src/models/`: Domain types and API structs.
    - `src/lib.rs`: WASM entry points and macro exports.
- **`.buildamp/macros/src/`**: Custom procedural macros (`BuildAmpEndpoint`, `BuildAmpContext`, etc.).

### Frontend (Elm & Vite)
- **`apps/web/src/`**: The main web application.
    - `Main.elm`: The Elm application entry point.
    - `Api.elm`, `Api/Schema.elm`: Generated Elm types (review for correctness of generation).
    - `index.js`: JavaScript entry point, handles WASM initialization and ports.
- **`apps/extension/src/`**: Chrome Extension source.
    - `background.js`: Background service worker.
    - `popup.js`, `popup.html`: Extension popup.

### Backend (Node.js)
- **`apps/server/`**: The backend server.
    - `server.js`: Main Express server, handles `hydrateContext` and DB connections.
    - `migrations/`: SQL migration files.

### Tooling
- **`packages/create-buildamp/`**: CLI tool for scaffolding new projects.
- **`packages/vite-plugin-buildamp/`**: Custom Vite plugin for WASM/Elm integration.
- **`scripts/verification/`**: Verification scripts (e.g., `verify_guest_identity.sh`).
