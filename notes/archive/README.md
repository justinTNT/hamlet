- `restructure-sprint1.md` - Sprint 1: Foundation planning
- `restructure-sprint3.md` - Sprint 3: CLI implementation
- `restructure-sprint3-complete.md` - Sprint 3 completion notes
- `restructure-sprint4.md` - Sprint 4: Migration planning
- `ARCHITECT_HANDOFF.md` - Handoff notes between sessions
